SEARCH ENGINE OPTIMIZATION

SEARCH ENGINE OPTIMIZATION

Abreviada para SEO, é um conjunto de técnicas que serve para otimizar os motores de busca.

Influenciam os algoritmos dos buscadores a definirem o ranking de uma para para determinada palavra chave que foi pesquisada.

Otimizar os mecanismos de busca é importante pois isso ajuda a alcançar bom posicionamento orgânico de páginas na web nos buscadores, entregando um conteúdo de qualidade e uma boa experiência para os usuários.

Rasteamento e a indexação

São processos necessários para que a informação fique organizada e acessível.

Rastreamento: o google rastreia as bilhões de páginas online que existem, utilizando códigos conhecidos como rastreadores ou bots.

Indexação: após a página ser rastreada o google identifica ela, e ai ocorre o processamento e analise do conteúdo. Nesse processo é avaliado se existe páginas duplicadas ou atributos alternativos. Após isos o google renderiza as páginas.

OTIMIZAÇÃO ON-PAGE

Otimização on page são todas as otimizações feitas dentro da página.

EXEMPLOS:

-Titulo com palavra chave principal

-Colocar descrição nas imagens do texto

-Inserir links para outros conteúdos do próprio site

-URL e subtitulos adequados

Práticas não indicadas para indexadores

Black Hat SEO ou seja táticas usadas para ranquear um site que violam as diretrizes dos mecanismos de pesquisa.

EXEMPLOS DE BLACK HAT:

-Texto Oculto

-Links pagos

-Redircionamentos enganosos

-Cloaking
